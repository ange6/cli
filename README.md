# @angel6/cli

## Install

```bash
$ npm i @angel6/cli -g
```

## Usage

```bash
# 显示帮助信息
$ angel -h

# 显示已支持的模板列表
$ angel list

# 检查cli版本号
$ angel check

# 生成项目结构
# template: 1、vue-webpack   // webpack打包的vue项目
$ angel init <template> <app-name>

```

## 说明

> `angel init` 作为项目初始化的脚手架命令，可以快速通过拉远程仓库的取模板创建一个新项目。
> 初始化阶段可选择项目依赖项。
